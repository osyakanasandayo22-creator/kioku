# 記憶力向上アプリ（HTML/CSS/JS）

スマホで使いやすい、白基調・余白多めのミニゲーム集です。  
外部サーバ不要で、端末内（localStorage）に履歴を保存します。

## 起動方法

1. このフォルダの `index.html` をダブルクリックで開く  
2. うまく開けない場合は、Chrome / Edge で `index.html` を開いてください

## できること（11ゲーム）

- ① 思考スナップショット回収装置：ランダムに「今何考えてた？」→直前ログと比較
- ② 意識ストリーム固定：テーマを決めて一定時間書き続ける→脱線ワード自己チェック
- ③ 短期記憶崩壊検知：列を記憶→遅延後に再現→長さ自動増減
- ④ ルール保持タスク：ルールを保持して数字に反応（条件一致時のみ）
- ⑤ 思考逆走テスト：文章を書いた後に「1文前/2文前…」を問う
- ⑥ ノールック編集：一瞬だけ表示→隠す→指示どおり編集して提出
- ⑦ 思考圧縮ループ：半分→さらに半分の文字数へ圧縮
- ⑧ 時間差自己対話：同じ質問に時間差で回答→差分可視化
- ⑨ マルチタスク分断耐性：A→割り込みB→Aへ復帰
- ⑩ 意図保持チャレンジ：意図入力→別タスク→意図再現
- ⑪ 英文字フラッシュカード：枚数選択→めくる→並べ替え→答え合わせ

## データについて

- 履歴は **localStorage** に保存されます
- 画面右上の「データ」から **エクスポート/インポート/全削除** ができます
